## Running the application

Use the below commands to run the application or use the IntelliJ 'Run' button.

Compile the program:

```bash
mvn compile 
```

Create the .jar package:

```bash
mvn package 
```

Run the app:
```bash
java -jar target/Tasks-1.0-SNAPSHOT.jar
```
